In this assignment we have to concatenate 2 files in a directory.
We first create a directory using the command "mkdir my_folder".
Then we change the directory using the command "cd my_folder".
We create 2 files using the command "vi my_file.txt" and "vi another_file.txt" and added the text on both file.
For concatenate second into first file we use the command "cat another_file.txt >> my_file.txt".
We check if this command works properly or not we check the content of my_file.txt using the command "cat my_file.txt".
It successfully added the text without overwriting it.
